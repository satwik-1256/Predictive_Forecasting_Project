# Predictive_Forecasting_Project
Interactive predictive forecasting dashboard for analyzing HHS Unaccompanied Children Program data using Machine Learning, ARIMA, and Streamlit.
# Predictive Forecasting Dashboard

## Project Overview

This project focuses on predictive forecasting and operational analytics using the HHS Unaccompanied Children Program dataset. The project uses machine learning and statistical forecasting techniques to analyze historical trends and forecast future care load demand.

An interactive Streamlit dashboard was developed to visualize forecasting performance, model comparison, KPI analysis, and operational forecasting insights.

---

# Objectives

- Analyze historical operational care load data
- Perform data preprocessing and feature engineering
- Build forecasting models for predictive analysis
- Compare machine learning and statistical forecasting approaches
- Develop an interactive Streamlit dashboard for visualization
